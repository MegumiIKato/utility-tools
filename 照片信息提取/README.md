# 照片GPS信息提取工具

## 功能说明

### 1. GPS信息提取
从照片中提取GPS经纬度和拍摄方位角信息，导出为CSV文件。支持自动扫描"人工举证照片"和"无人机举证照片"两个文件夹，分别生成对应的CSV文件。

### 2. 照片批量重命名
将指定目录下所有子文件夹内的照片，按照`文件夹名后五位-序号`格式批量重命名。

示例：
- 文件夹 `330Z060054` 内的照片 → `60054-1.jpg`、`60054-2.jpg` ...
- 文件夹 `330Z060055` 内的照片 → `00055-1.jpg`、`00055-2.jpg` ...

## 使用方法

### GPS提取
1. 在「GPS提取」区域点击"浏览..."选择包含照片文件夹的目录
2. 点击"开始提取"按钮
3. 等待处理完成，CSV文件将保存在所选目录下

### 照片重命名
1. 在「照片重命名」区域点击"浏览..."选择包含子文件夹的照片目录
2. 点击"照片重命名"按钮
3. 确认后开始重命名

## 支持的图片格式
- JPG/JPEG
- PNG
- TIFF
- HEIC

## 输出格式

### GPS提取CSV
CSV文件使用UTF-8编码（带BOM），包含四列：
- filename: 图片文件名
- latitude: 纬度（度格式）
- longitude: 经度（度格式）
- azimuth: 拍摄方位角（度，0°=正北，顺时针增加）

### 生成的文件
- `人工举证照片坐标信息.csv`
- `无人机举证照片坐标信息.csv`

# Utility Tools

一系列独立的小工具集，用于提高日常工作效率。

## 工具列表

| 工具 | 说明 | 平台 |
|------|------|------|
| [照片GPS信息提取](照片信息提取/) | 从照片EXIF中提取GPS坐标并导出CSV | Windows |

## 使用方法

每个工具都是独立的，进入对应目录查看详细说明。

## 打包EXE

本仓库使用 GitHub Actions 自动打包：

1. 进入 [Actions](../../actions) 页面
2. 选择 **Build EXE**
3. 点击 **Run workflow**
4. 等待完成后在 **Artifacts** 下载 exe 文件
